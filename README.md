# Network Automation Lab

This project demonstrates basic network automation using Python.

The script connects to a Cisco switch via SSH and automatically applies configuration commands such as VLAN creation and hostname modification.

## Features

- SSH connection to Cisco switch
- Automated VLAN configuration
- Hostname modification
- Automatic configuration saving

## Technologies Used

- Python
- Netmiko
- SSH

## How to Run

Execute the script using:

py switch_config.py

## Purpose

This project demonstrates how network engineers can automate repetitive configuration tasks using Python and network automation libraries.
